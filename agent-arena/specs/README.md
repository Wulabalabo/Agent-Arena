# Agent Arena Specs

These specs define the next Agent Arena MVP direction before a development plan is written.

Read order:

1. `01-product-spec.md`
2. `02-frontend-page-spec.md`
3. `03-predict-integration-spec.md`
4. `04-agent-workshop-spec.md`
5. `05-data-state-and-acceptance-spec.md`

The current product direction is Predict-native:

- Users back Agents in an arena experience.
- Sui Predict is the underlying market, position, and settlement protocol.
- The MVP must avoid building a custom prediction-market protocol.
- Agent Workshop is a mock/demo surface in the MVP.
